# next-p5js
## a dev env for p5js using nextjs

### features
  - sketches explorer (gallery)
  - sketches options
  - recorder (video and screenshots)
  - social network sharer (twitter, nft, instagram)
  - sketches exporter (github pages, html, video)
  - nft creator
  - cli?, or snipepts
  - utils
    - midi
    - audio capture
  
## orther compatible libs
  - three.js
  - swissGL
  - paperjs
  - motion etc...
